# 🎓 Student Placement Prediction Using Machine Learning

## 📌 Project Overview

This project predicts whether a student is likely to be placed based on academic performance, internship experience, projects, certifications, aptitude score, and skill assessments.

The system applies machine learning techniques to analyze student profiles and generate placement predictions. Multiple classification algorithms were evaluated, and Random Forest was selected as the final model due to its strong performance and stability.

---

## 🎯 Problem Statement

Educational institutions often lack a data-driven method to evaluate student placement readiness. Students may not know whether their current academic and skill profile is sufficient for placement opportunities.

This project aims to address this challenge by developing a machine learning model that predicts placement outcomes using student-related attributes.

---

## 📊 Dataset Features

The model uses the following features:

- CGPA
- Internship Experience
- Number of Projects
- Certifications
- Aptitude Test Score
- Communication Skills
- Technical Skills
- Internship Flag

---

- Python
- Pandas

- Joblib

---

## 🤖 Machine Learning Models Evaluated

The following algorithms were tested and compared:

- Logistic Regression
- Decision Tree Classifier
- K-Nearest Neighbors (KNN)
- Support Vector Machine (SVM)
- Random Forest Classifier

---

## 🏆 Final Model

**Random Forest Classifier**

Reasons for selection:

- Better overall performance
- Handles non-linear relationships effectively
- More robust than individual decision trees
- Stable prediction behavior
